アカウントを削除する

アカウントの削除に関する情報を記載する．

  • SMS 認証は一定期間経過後できるようになる．
  • アカウントを削除する際は，フォロー・フォロワーを 0 にする
  • プロフィールやアイコンも修正しておいたほうが良いかも
SMS 認証

SMS 認証を行ったアカウントを削除し， 新たにアカウントを作成した場合，SMS認証を再度できるか？という問題がある．

公式に問い合わせた結果， 「一定期間経過後 SMS 認証はできるようになる．ただし，その期間は非公開である」 という回答を得た．

（筆者は，以下の影響から，翌々月以降に SMS 認証ができるようになると想像している．） トップファン

リスナーとして記録に残るのが「トップファン」である． 前月分まで掲載されるので，翌月末までアカウント名が掲載される． ギフトを贈らなくても，スーパーライク一回からアカウント名が掲載される点に注意されたい．

トップファンには，「ファンバッジ」と「ギフト」の２つがあるが， それぞれ削除されたアカウントに対する挙動が異なる．

「ファンバッジ」一覧は，アカウントを削除するとすぐにみえなくなる．

一方，「ギフト」一覧は，アカウントを削除しても名前が残る．

イベント

ライバーとして記録に残るのが「イベント」である． イベントは「過去のイベント」に２ヶ月残る．

なお，著者は，過去イベントでの削除されたアカウントの挙動を確認していない．

フォローとフォロアー

フォローされた側・フォローした側からの表示に削除されたアカウントはそのまま残る．

やっかいなことに，2025/08/13 現在， 削除されたアカウントのフォロー・フォロワーは， アカウントを表示しようとすると，「退会したユーザー」と表示されブロックできない． （遠くない将来修正されると期待している．）

そのため，アカウントの削除を検討しているユーザーは， フォロー・フォロワーを 0 にしてからアカウントを削除することをお勧めする． なお，フォロワーを 0 にするには，ブロックすることで実現できる． ご参考
